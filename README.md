table format in HTML:
<table>
  <thead>     //this represents the top row of a table, it has the headers for each column
    <tr>      //table row, starts a new row
      <th>HEADING</th>
    </tr>
  </thead>
  <tbody>     //starts the table body
    <tr>
      <td>DATA</td>
    </tr>
  </tbody>
</table>
